Overview

What is clAIre?

The Problem

In Lake County, thousands of seniors living alone face growing safety and health challenges. Falls, delayed emergency response, and isolation are among the most serious — and without support, everyday situations can quickly become emergencies.

Our Solution

clAIre (pronounced Claire) is an autonomous robotic companion that combines real-time obstacle detection, voice-activated emergency response, and autonomous following into one accessible, always-on assistant.

clAIre doesn't replace caregivers — it fills the gaps between them.

Capabilities

Key Features

🔊
Verbal Obstacle Warnings

Detects and announces obstacles in real time: "Obstacle ahead — stairs on the left."

🚨
Voice Emergency Calling

Say "Call for help" and clAIre contacts emergency services immediately with location data.

🧍
Autonomous Following

Tracks and follows the user at a safe distance (~1–2 meters), completely hands-free.

🎙️
Natural Voice Commands

Fully hands-free operation — no buttons, no screens required. Always-on wake word detection.

📱
Companion App

Dashboard with live alerts, reminders, health info, battery status, and activity logs for caregivers.

🔋
All-Day Battery

Dual 5,200 mAh lithium batteries — estimated 8–12 hours of continuous operation.

Technical

System Architecture

Display & Interaction
Raspberry Pi — system coordination & voice AI
Hamtysan Monitor — visual interface
Dual speakers — audio output / verbal feedback
Microphone — voice command input
Navigation & Vision
Pi Camera Module — top-mounted, obstacle detection & user tracking
Navigation ESP32 — vision & obstacle processing
ESP-NOW wireless — low-latency inter-ESP communication
Motor Control
Main ESP32 — behavioral & motor control
PCB Motor Driver — drive logic
550 35T Brushed motor — main propulsion
370 Planetary motors (×2) — drive support
Power System
2× 5,200 mAh Li-Ion batteries (~10,400 mAh total)
2× Crawler BDC80 ESCs — power/discharge management
Over-discharge protection — independent L/R channels
Estimated runtime: 8–12 hours continuous

Behavior

Operational Modes

🚶 Normal Following
  1. Camera continuously tracks user
  2. Nav ESP32 calculates movement
  3. Maintains 1–2 m safe distance
  4. Speed auto-matches user pace
  5. Verbal confirmations keep user informed
🚧 Obstacle Detection
  1. Camera identifies obstacle
  2. Classifies type and location
  3. Immediate STOP command to motors
  4. Announces: "Obstacle ahead — [type] at [location]"
  5. Waits for clearance or command
🆘 Emergency Response
  1. User says "Call for help"
  2. Raspberry Pi processes command
  3. Confirms: "Calling emergency services now"
  4. Emergency call initiated with location
  5. Audio connection maintained
🎙️ Voice Interaction
  1. Always-on wake word detection
  2. Natural language processing
  3. Verbal + visual confirmation
  4. No buttons or screens required
